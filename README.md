# Bug Trap
This  is a short guide to writing code with less bugs. It is written as an
ebook generated by Doxygen with example code in C++. The book
covers general principles, but the examples are taken from C++.

## Bugtrap: How to code with fewer bugs

Below you find a link to a ready-built e-book:
[Bugtrap e-book](https://github.com/harald1984/bugtrap/master/html/index.html)

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

```
* Doxygen
* gcc
* gnu make
* googletest
* lcov
* valgrind
```

### Building

```
> make all
```
